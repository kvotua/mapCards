# Требования к 3D-моделям для проекта

В данном документе описаны ограничения и рекомендации для 3D-моделей, используемых в проекте. Модели отображаются на карте с помощью библиотеки Three.js и компонента `GLView` из `expo-gl`

---

## 1. Поддерживаемые форматы

- `GLB (.glb)` - предпочтительный формат
- `GLTF (.gltf)` - не рекомендуется, так как требует внешних файлов, что усложняет загрузку в Expo
- Другие форматы нельзя загрузить напрямую через `GLTFLoader` в текущей конфигурации проекта

В будущем возможно добавление поддержки других форматов моделей (например, `OBJ`) с использованием дополнительных загрузчиков (`OBJLoader` и других).

---

## 2. Материалы

Так как модели загружаются динамически с использованием `expo-three`, типы материалов определяются на этапе экспорта из 3D-редактора. В процессе загрузки не происходит переопределения материалов в коде.

Поддерживаются только стандартные материалы, совместимые с Three.js в Expo (к примеру `MeshStandardMaterial`, `MeshBasicMaterial`).

Все текстуры должны быть встроены в `.glb` файл.

---

## 3. Ограничения по весу

- Максимальный размер `.glb`: `1MB`
- Рекомендуемый размер: `≤ 400KB`

---

## 4. Импорт моделей в проект

**! Один файл `.glb` на одну модель**

Для подключения 3D-моделей к маркерам на карте необходимо выполнить следующие шаги:

1. Скопируйте `.glb` файлы в папку:

`assets/models3D`

2. Откройте файл:

`src/entities/marker/model/markers.ts`

3. В каждом объекте маркера укажите путь к модели в поле `model` с использованием `require`. Пример:

```ts
export const markers = [

    {

        key: '1',

        title: "Музей янтаря",

        description: "Музей янтаря расположен в башне Дона — уникальном историческом здании. Здесь можно увидеть невероятные образцы янтаря с вкраплениями древних насекомых и растений, украшения и предметы искусства, изготовленные из янтаря.",

        address: "Калининград, пл. Маршала Василевского, 1",

        model: require('../../../../assets/models3D/Avocado.glb'),

        icon: require('../../../../assets/markers_icon/Avocado.png'),

    },

]
```
